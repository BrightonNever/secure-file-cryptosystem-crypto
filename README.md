# 🔐 Secure File Cryptosystem (OpenSSL Implementation)

## 📌 Overview
This project demonstrates the practical implementation of core cryptographic security principles using OpenSSL in a Linux environment. It simulates a secure file protection workflow incorporating encryption, digital signatures, and integrity verification.

Developed as part of advanced cybersecurity coursework, this project reinforces applied cryptography concepts used in enterprise security systems.

---

## 🔒 Security Concepts Implemented

- SHA3-256 Hashing (Integrity)
- ECDSA Digital Signatures (Authentication & Non-Repudiation)
- AES-256-CBC Encryption (Confidentiality)
- Public/Private Key Generation
- Secure File Verification Workflow

---

## ⚙️ Features

- Generate ECDSA key pairs
- Sign files using SHA3-256
- Verify digital signatures
- Encrypt files using AES-256
- Decrypt encrypted files

---

## 🛠 Technologies Used

- Bash Scripting
- OpenSSL Toolkit
- Linux Environment (WSL/Ubuntu)
- Public Key Cryptography (ECC)
